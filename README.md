# Pacman Game (Java)

This is a simple console-based Pacman game developed in Java. The player controls Pacman and moves inside a maze while collecting food and avoiding the ghost.

## Features
- Maze-based game board
- Player (Pacman) movement using W/A/S/D keys
- Random ghost movement
- Score tracking system
- Game Over when ghost catches Pacman

 Game Logic
- Pacman starts at position (1,1)
- Ghost moves randomly in the maze
- Player collects food (.) to increase score
- Walls (#) block movement
- Game ends when Pacman and Ghost collide
